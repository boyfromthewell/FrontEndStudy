# HTML (Hypertext Markup Language)

복습 겸 마크다운 문서 연습하기<br>


* Block - 충분한 공간 있어도 한 줄에 하나 (div)
* Inline - 공간 있으면 삽입 가능 (p, span)

## box형
```html
<header></header>
<footer></footer>
<section></section>
<div></div>
<span></span>
```
## item형
```html
<h1>hi</hi>
<button></button>
```
* a 태그 - 링크
## ol, ul, li
```html
<ol>
    <li>순서 있는 리스트</li>
</ol>

<ul>
    <li>순서 없는 리스트</li>
</ul>    
```
